⚙️ Configuración inicial (una sola vez)

Para conectar Gmail necesitas crear credenciales en Google Cloud. Sigue estos pasos:

1
Ve a Google Cloud Console
console.cloud.google.com → Crea un proyecto nuevo → Llámalo "CICSA"
2
Habilita la Gmail API
APIs & Services → Enable APIs → busca "Gmail API" → Enable
3
Crea credenciales OAuth
APIs & Services → Credentials → Create Credentials → OAuth Client ID → Desktop App → Descarga el JSON
4
Pon el archivo en la carpeta del programa
Renómbralo a gmail_credentials.json y cópialo junto a INICIAR_CICSA.bat
5
Primera autorización
La primera vez que hagas clic en "Revisar Gmail", se abre el navegador para que autorices el acceso a tu cuenta. Solo ocurre una vez.
